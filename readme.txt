In this project you can search any items in table by select field in select element, 
	then enter the value in search panel and click on button "Найти"

Also you can sort all data in table by click on table head cells, data sorting by asceding and desceding
If you click on any table row you can see information about this person after the table
And for your navigation you can use arrows button after the table, to load new data

For open project localy you need clone this repo
	then cd to project
	then npm i 
	then npm run dev
	then in yout browser open http://localhost:8080/

To see project remote you can visit - https://dimashpk.github.io/Table
